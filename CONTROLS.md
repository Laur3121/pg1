# 操作方法 (Controls)

## 基本操作 (Basic Controls)
- **移動 (Move)**: 矢印キー (↑ ↓ ← →) / Arrow Keys
- **会話・インタラクト (Interact)**: F キー / F Key

## メニュー操作 (Menu Controls)
- **バッグ (Inventory)**: B キー / B Key
- **ステータス (Status)**: C キー / C Key
- **セーブ (Save)**: S キー / S Key
- **デバッグメニュー (Debug)**: D キー / D Key
- **クエストログ (Quest Log)**: Q キー / Q Key
- **ショップ (Shop)**: P キー / P Key
- **メニューを閉じる (Close)**: ESC キー / ESC Key

## 戦闘 (Battle)
- **形式**: オートバトル (Automatic Battle)
  - キャラクターは自動的に行動します。
- **デバッグ (Debug - Mouse Click)**:
  - **Win**: 戦闘に強制勝利します。
  - **Lose**: 戦闘に強制敗北します。
